# CAN Based IDS Proposal
## 23-1 빅데이터응용보안 4조

This project is for detecting anomaly messages from CAN based data set.
- method: machine learning, deep learning
- model: SVM, KNN, RF, AdaBoost, IF, CNN
- data: Car-Hacking Dataset from HCRL
